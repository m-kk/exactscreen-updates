# ExactScreen Updates

Public Sparkle appcast and release binaries for [ExactScreen](https://github.com/m-kk/exactscreen).

- Feed: `https://m-kk.github.io/exactscreen-updates/appcast.xml`
- Binaries: attached to this repo's GitHub Releases.

The `appcast.xml` is generated by Sparkle's `generate_appcast` over signed
release archives and committed here; GitHub Pages serves it over HTTPS.
